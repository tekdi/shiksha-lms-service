# Features

The LMS service provides a comprehensive suite of features for modern digital learning management.

### Course Management
- **Structured Content**: Create courses with titles, descriptions, and thumbnails.
- **Organization**: Group courses by tenant and organization.
- **Pricing & Rewards**: Configure course pricing and completion rewards (Certificates, Badges).
- **Prerequisites**: Define required courses that must be completed before starting a new one.
- **Workflow**: Manage course publishing states (Published, Unpublished, Archived).

### Module & Lesson Architecture
- **Hierarchical Modules**: Break down courses into manageable modules and submodules.
- **Diverse Lesson Types**:
  - **Video**: Support for YouTube, Vimeo, and direct video URLs.
  - **Document**: Integration with PDF and external document links.
  - **Assessment**: Built-in quiz and assessment capabilities with passing criteria.
  - **Event**: Integration with online and offline events.
  - **Text & Media**: Flexible formatting for combined content.
- **Ordering**: Customizable sequence for modules and lessons.

### Enrollment & Progress
- **Flexible Enrollment**: Support for manual, administrative, and auto-enrollment.
- **Access Control**: Expiry-based access management with automated mail notifications.
- **Granular Tracking**:
  - **Course Track**: Overall progress within a course.
  - **Module Track**: Completion status at the module level.
  - **Lesson Track**: Detailed tracking including attempts, scores, and time spent.
- **Resume Capability**: Allows learners to pick up exactly where they left off.

### Platform Features
- **Multi-tenancy**: Complete data isolation between different tenants.
- **Globalization**: Built-in support for internationalization (i18n).
- **Scalability**: High-performance backend built on NestJS and PostgreSQL.
- **Caching**: Optimized performance with Redis-based caching.
