# About

The LMS (Learning Management System) microservice is a central component of the Shiksha platform, designed to manage educational content and track learner progress. It provides a robust and scalable solution for organizing courses, modules, and lessons, while also handling user enrollments and comprehensive progress tracking.

Built with NestJS, the service offers a high-performance, modular architecture that supports multi-tenancy, allowing different organizations to manage their own educational content in isolation. 

Key capabilities of the LMS service include:
- **Hierarchical Content Organization**: Courses are organized into modules, which can contain submodules and individual lessons.
- **Versatile Lesson Formats**: Supports various content types including video, documents, assessments, and events.
- **Dynamic Progress Tracking**: Monitors learner advancement at every level of the course hierarchy.
- **Flexible Enrollment Management**: Handles user access to courses with support for administrative approvals and auto-enrollment logic.
- **Data Isolation**: Ensures secure and isolated data for multiple tenants and organizations.
